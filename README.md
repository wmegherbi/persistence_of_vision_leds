# persistence_of_vision_leds
Ce projet a pour objectif de concevoir et réaliser un système d’affichage basé sur la persistance rétinienne sur une barre de LEDs en rotation. En exploitant la persistance rétinienne de l’œil humain, le système permet d’afficher des images fixes ou animées dans l’espace à partir d’un nombre limité de LEDs disposées sur une structure rotative.


Ce projet se divise en trois parties différentes : 
- Partie Mécanique : plans et schémas du système complet ainsi que des impressions 3D 
- Partie LEDs : contrôle, alimentation des LEDs et traitement d'image via la Raspberry Pi située dans la partie rotative
- Partie Moteur : contrôle, alimentation du moteur et mise en place de l'interface de contrôle du moteur via la Raspberry Pi située dans la partie statique
